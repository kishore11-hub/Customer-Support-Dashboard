# Customer Support Dashboard (Power BI)

## Project Overview

This project analyzes customer support operations using Power BI. The dashboard tracks ticket volume, issue trends, resolution performance, and customer satisfaction.

## Dataset

* 500+ customer support records
* Fields include: Ticket ID, Date, Channel, Issue Type, Priority, Status, Resolution Time, CSAT

## Key Insights

* Total Tickets and Open vs Closed status
* Issue category analysis
* Channel-wise ticket distribution (Email, Call, Chat)
* Average resolution time
* Customer satisfaction (CSAT)

## Tools Used

* Microsoft Excel
* Power BI

## Purpose

To monitor customer support performance and identify areas for service improvement.
